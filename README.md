# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/130lq0XR3qrVTyje8-wX0Q7nqvcCAdQ3J

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

# 注意：来自作者本人的特别说明

本来我选择了一个非常适合的BGM作为游戏中的背景音乐，但由于多次修改一直没有可行的办法实现程序自动播放我上传好的BGM，无奈出此下策。

我将会添加一个“BGM”文件夹，在里面有我已经选好的背景音乐音频文件（在最终提交的作业文件夹内也会有一份这样的文件方便老师上传体验）。在游戏首页有一个“音乐设置”界面，在此界面将文件手动上传（程序运行期间仅需上传一次，重启程序需要重新上传）。

当然，这个偶然的设计也意外地增加了游戏的可玩性，你可以上传自己的音乐，并可以自由调节音量和播放开关，就当作是赶DDL时听着你专属的专注歌单吧……
